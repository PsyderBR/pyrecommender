## Sobre

Este projeto visa ajudar as pessoas a calcular seu Índice de Massa Corporal (IMC), uma medida simples para avaliar o peso corporal em relação à altura.

## Começando

Para executar este projeto localmente, siga as instruções abaixo:

### Pré-requisitos

- Python 3 instalado

### Instalação

1. Clone este repositório: `git clone https://github.com/seu_usuario/calculadora-imc.git`
2. Navegue até o diretório do projeto: `cd calculadora-imc`
3. Execute o aplicativo: `python calculadora_imc.py`

## Uso

Execute o script Python e siga as instruções para inserir seu peso e altura. O aplicativo calculará automaticamente seu IMC e mostrará o resultado.
